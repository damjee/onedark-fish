# One Dark colors for [fish shell](https://fishshell.com/)

## Installation

Install themes with [`fisher`](https://github.com/jorgebucaran/fisher):

```sh
fisher install woheedev/onedark-fish
```

Choose the theme:

```sh
fish_config theme save "One Dark"
```
